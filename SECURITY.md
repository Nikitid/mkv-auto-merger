# Безопасность данных

Этот проект не обрабатывает учетные данные, но выполняет операции с
пользовательскими файлами.

Перед использованием:

- сделайте резервную копию незаменимых файлов;
- запустите `DRY RUN` и проверьте все пути;
- не используйте режим удаления, пока результат не проверен;
- не запускайте скрипт с избыточными правами;
- не публикуйте журналы, если имена файлов содержат личные данные.

Ошибки, способные привести к перезаписи, удалению или перемещению неверных
файлов, следует сообщать через GitHub Issues без прикладывания пользовательской
медиатеки.
